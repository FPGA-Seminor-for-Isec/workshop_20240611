# 課題03：FPGAにブザーをつないで、好きなメロディーを奏でよう

[トップに戻る](../../README.md)

## 概要

FPGA上にあるGPIOピンにブザーを接続し、自由に音楽を鳴らしてみよう。

## 詳細

### 期待する動作

- FPGAにつないでいるブザーを使って、思い通りの音楽を鳴らすこと。

### HDLの仕様

```
入力：必要に応じて使用すること
出力：pin_fpga_gpio[1]
```

### 回路図

![課題03の回路図](./figure/ex03_schematic.png)

### 注意事項

- FPGAのGPIOピンに電圧を与えたり、ショートするような回路を設計しないこと。（FPGA側に電流が流れてこわれることがあります。）
